# Telegram Catalog Bot — Specification

## 1. Цель проекта

Нужно реализовать Telegram-бота-каталога для продажи товаров из прайс-листа.

В каталоге есть:

- категории товаров;

- товары внутри категорий;

- артикулы;

- наименования;

- цены;

- в перспективе — изображения для каждого артикула.

Пользователь должен удобно просматривать каталог, искать товары, добавлять позиции в заявку и отправлять заявку администратору Telegram-бота.

На первом этапе не требуется:

- онлайн-оплата;

- полноценный личный кабинет;

- сложная CRM;

- web-интерфейс администратора.

Основной сценарий — пользователь выбирает товары, бот отправляет заявку администратору.

---

## 2. Технологический стек

Использовать:

- Node.js

- TypeScript

- PostgreSQL

- Telegram Bot API

- Docker / Docker Compose

- dotenv для конфигурации окружения

Рекомендуемые библиотеки:

- `telegraf` или `grammy` — Telegram bot framework

- `pg` или `prisma` — работа с PostgreSQL

- `dotenv` — загрузка переменных окружения

- `zod` — валидация конфигурации и входных данных

- `pino` или `winston` — логирование
